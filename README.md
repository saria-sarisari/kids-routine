# kids-routine ⏰

子どもの朝の支度をサポートする、アナログ時計ベースの時間管理ツールです。  
ASDのある小学校低学年（6〜7歳）を主な対象として、視覚的にわかりやすい設計にしています。

## デモ

> リンクをここに貼る

## 機能

- **アナログ時計（秒針つき）** — 時間が進む様子をリアルタイムで確認できます
- **デジタル時計** — 針が読めなくても時刻がわかります
- **予定の色づけ** — やることと時間帯を登録すると、時計の内側が色で塗られます
- **複数予定対応** — 「あさごはん」「おしたく」など複数の予定をそれぞれ別の色で表示
- **自動リセット** — 時間が過ぎた予定は自動で色が消えます
- **いまどこ？バッジ** — 現在時刻の予定名が大きく表示されます

## 使い方

1. 「なにをする？」に活動名を入力（例：あさごはん）
2. はじまりとおわりの時間を選ぶ
3. 「🌟 ついか！」ボタンを押す
4. 時計に色つきの扇形が表示される

## 対象

- ASDのある小学校低学年（6〜7歳）を主な想定
- 朝の支度ルーティンの可視化・時間感覚の育成を目的としています
